# Vietnam Travel simple CRUD Web App

A backend-focused CRUD web application for managing Vietnam travel destinations.

## Tech Stack
- Node.js
- Express.js
- MSSQL
- EJS (View Engine)
- MVC architecture

## Features
- Create / Read / Update / Delete travel destinations
- Duplicate checking
- Input validation
- RESTful API design
- SweetAlert2 UI feedback

## Run locally
```bash
npm install
npm start
